# HTML Template & Article Management System

## Overview

This is a professional, Joomla-style content management system that allows you to:
- **Upload HTML template files** and automatically extract editable positions
- **Create and manage articles** organized by categories
- **Assign articles to templates** and populate specific content positions
- **Use rich text editor** (Quill.js) for content creation
- **Color-coded categories** for better organization
